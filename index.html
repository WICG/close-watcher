Close signals and close watchers are moving to the HTML Standard!

The best specification for close signals, close watchers, and the CloseWatcher API currently lives as a draft pull request against the HTML Standard: whatwg/html#9462. You can also view a PR preview. Compared to previous versions that lived at this URL, a few bugs were fixed, and several monkeypatches were moved to the appropriate location.

Once that PR is merged, this URL will redirect to the final specification.
